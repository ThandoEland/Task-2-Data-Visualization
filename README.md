# Task 2 - Data Visualization and Storytelling

## Objective
To create a Tableau dashboard that presents business insights using the Sample Superstore dataset.

## Tools Used
- Tableau Public
- Sample Superstore Dataset
- GitHub

## Visualizations Created
1. Sales by Category
2. Profit by Region
3. Monthly Sales Trend
4. Sales by Sub-Category
5. Profit vs. Sales (Scatter Plot)

## Key Business Insights
- Technology generated the highest sales.
- The West region recorded the highest profit.
- Sales changed throughout the year, with noticeable peak periods.
- Some sub-categories contributed more to sales than others.
- High sales do not always result in high profits.
